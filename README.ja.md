# PonDeStyle

PonDeStyleは、Flexboxを用いたシンプルで新しいCSSフレームワークです。PostCSSで開発されています。

## サポートブラウザ

* IE11+
* Firefox* (Desktop, Android)
* Chrome* (Desktop, Android)
* Safari* (Desktop, iOS)
* Edge*

\* 2バージョン前まで
